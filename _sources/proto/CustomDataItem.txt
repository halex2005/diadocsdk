CustomDataItem
==============

.. code-block:: protobuf

    message CustomDataItem
    {
        required string Key = 1;
        optional string Value = 2;
    }

Структура представляет произвольную пару "ключ-значение", привязанную к документу в качестве пользовательских данных.

-  *Key* - произвольный непустой регистронезависимый ключ. Может содержать только буквы латинского алфавита, цифры и символы "-" и "_". Длина не может превышать 128 символов.
-  *Value* - произвольное регистронезависимое значение (возможно, пустое), соответствующее ключу Key. Длина не может превышать 1024 символа.
